    Detailed Description

    The widget that used for rendering painting results.

    Canvas uses several QPixmap to render contents. Each layer opaque one QPixmap and painted together when paintEvent happens.

    Canvas can be used for mouse painting, tablet painting and remote painting via json messages from network.

    Brush is used for painting. There're 4 kind of brush currently and will be more. When changing color of a Brush using on Canvas now, you can use either setBrushColor() or setColor() .

    See also Brush.
